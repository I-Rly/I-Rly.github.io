"Hasher"

Post graduate programming project

"Hasher" is a simple Python commandline tool for calculating the MD5 and SHA1 hashes of files. "Hasher" is able to detect protential MD5 and/or SHA1 hash collsions (where two files have the same MD5 or SHA1 hash) and report them as such. It is able to compare two files or one file against a folder. The program produces seperate timestamped TXT files for logging and results reporting.
